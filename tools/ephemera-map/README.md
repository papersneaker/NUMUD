# Ephemera Map Tool

The Ephemera Map Tool is a self-contained, interactive HTML visualisation of the city of Ephemera's surface layer, spanning all 42 districts. It renders a clickable grid-based city map with per-district metadata (name, faction, status, and flavour text) drawn entirely from data embedded in the file itself — no server, no build step, and no external dependencies beyond the Google Fonts stylesheet loaded at runtime. To open it, simply double-click `ephemera_map.html` in your file manager or drag it into any modern browser; the full map loads instantly as a static page. The **Export** button serialises the current district dataset to a timestamped JSON file and triggers a browser download, giving you a portable snapshot of the map state suitable for backups, handoffs, or import into other tooling. The embedded district data constitutes the canonical surface-layer reference for Ephemera and should be treated as the authoritative source of record until a backend data service is connected and the map is updated to fetch from it instead.
